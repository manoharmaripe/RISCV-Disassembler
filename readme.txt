This folder consists of the code for given question named as Lab3.c, a report and some input files

To compile the code  open the terminal and navigate to this directory then run the following command in the terminal
gcc Lab3.c

To execute run the command in the terminal
./a.out

After this the terminal waits for the input from the user, user will be asked to provide the name of input file,

If the input file is in this folder then give it's name directly 
For example
input.txt

Otherwise give the complete path
For example 
/Users/manoharmaripe/Desktop/input2.txt

Enter path of input files and click enter and then the assembly code of given dissembled code will be displayed as output.

It is considered that the input file contains 32 bit instructions in hexadecimal format(without 0x in the start for example 007201b3)
